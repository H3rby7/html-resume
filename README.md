# Getting Started

    npm i
    npm start

Edit to your liking and then [print](#printing).

## Printing

1. Print Page (CTRL+P)
2. Select 'Margins' > None
3. Uncheck 'Print header and footer'
4. Check 'Print background'
5. OPT: Set 'Scale' to 100
